# SAP ABAP Payroll Employee Management System

## Project Overview
The SAP ABAP Payroll Employee Management System is developed to automate employee management, salary management, and payroll processing. It helps in maintaining payroll records efficiently and reduces manual work.

## Features
- Employee Management (Add, Display, Update, Delete)
- Salary Management (Add, Display, Update, Delete)
- Payroll Processing
- Payroll Report Generation
- Main Navigation using Module Pool Programming

## Technologies Used
- SAP ABAP
- Module Pool Programming
- Screen Painter
- Menu Painter
- Open SQL
- Dictionary Objects
- Internal Tables
- Work Areas
- Selection Screens

## Database Tables
- ZPMS_EMP_MST (Employee Master Table)
- ZPMS_SAL_MST (Salary Master Table)
- ZPMS_PAY_TRN (Payroll Transaction Table)

## Reports
- ZPMS_EMPLOYEE_MGMT
- ZPMS_SALARY_MGMT
- ZPMS_PAYROLL_PROCESS
- ZPMS_PAYROLL_RPT

## Transaction Code
ZPAYROLL
